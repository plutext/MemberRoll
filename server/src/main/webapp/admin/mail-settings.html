Mail settings

How the society sends email — payment receipts, lost-link replies and segment sends all use this relay. Settings saved here take effect on the very next message; if you save nothing, the server's environment configuration is used (the bootstrap/dev fallback).

Microsoft 365 notes
  • The username must be the full mailbox address (e.g. noreply@society.org.au).
  • The From address must be that mailbox, or one it has Send As rights over.
  • The mailbox needs Authenticated SMTP enabled in the Microsoft 365 admin center (off by default on newer tenants).
  • Basic (password) auth works through December 2026; from then it is default-disabled per tenant (re-enableable). OAuth is a later follow-up.
