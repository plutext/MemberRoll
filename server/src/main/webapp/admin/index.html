      The household-level default for each communication type; a person's own exception (in their record) overrides it. A greyed row is EMAIL by default.

      Renewals

      Run a membership year: open a period with its prices, roll the prior year's paid members forward, record the cash and transfers as they arrive, and export the AGM and mailing lists. Payments are insert-only — a mistake is fixed with a Reverse (an equal-and-opposite entry), never an edit.

      Reconciliation export

      Export categorised payments for a window (each payment's allocation split in columns, plus net totals), book the Stripe side with one clearing-account journal, then mark them reconciled so next month starts where this one ended. See the user manual for the journal procedure.

      Xero account mapping

      Opaque account codes from your Xero chart of accounts — the server never interprets them; a wrong code fails at Xero's import. The journal export stays dormant until these are saved.
