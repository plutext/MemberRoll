Import members (CSV)

A one-off bulk load — usually to seed a new install from the society's existing spreadsheet. For day-to-day changes use the register and renewals pages instead.

Massage your spreadsheet into the columns of docs/import-template.csv and export it as CSV UTF-8 (plain "CSV" from Excel mangles accented names). Preview is a dry run; Apply writes everything in one transaction, and is enabled only once a preview comes back with no errors.
