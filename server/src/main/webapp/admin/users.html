Self-serve provisioning

Creates (or adopts) a Keycloak account for every current formal member with an email address and links it to their register record (CR-006). Idempotent — run it after an import, a walk-in, or on a whim; already provisioned members report Already linked. Provisioning sends nothing: members opt in via the login page's Forgot Password.
