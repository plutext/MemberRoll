# Fork gap analyses

Gap analyses from (or for) forks of MemberRoll, contributed under the
rules in `docs/fork-philosophy.md`: the analysis is a contribution even
when the code it led to stays in a fork. Each doc teaches the next
similar organisation's agent the domain — regulatory targets, which
seams fit, where the bespoke risk concentrates — for free.

An entry should state: what the organisation is, which of its governing
documents drove the analysis, what transfers unchanged, the proposed
fork CRs, and an honest note on what was *not* verified.

| Doc | Organisation type | Governing documents |
|---|---|---|
